## 🤖 НАСТРОЙКА TELEGRAM БОТА (5 МИНУТ)

### Шаг 1: Создать бота
1. Открой @BotFather в Telegram
2. Команда: `/newbot`
3. Введи имя и username
4. Скопируй TOKEN

### Шаг 2: Добавить кнопку игры
1. В @BotFather: `/setmenubutton`
2. Выбери свой бот
3. Нажми Web App
4. Введи URL твоей игры:
   - `https://ТВОЙ_НИКИ.github.io/hook-butcher/`
   - или `https://hook-butcher.vercel.app`
5. Текст: `🔪 Играть`

### Шаг 3: Тестировать
1. Найди своего бота в Telegram
2. Пишешь `/start`
3. Видишь кнопку "Играть"
4. Кликаешь → открывается игра

**ГОТОВО! 🎉**

## 🚨 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Игра не загружается
- Проверь консоль (F12 → Console)
- Убедись что Phaser CDN доступен
- Попробуй другой браузер

### Нет звуков
- Включи звук на телефоне
- В браузере разреши звуки (иногда требует клик)
- Проверь громкость в системе

### Сохранения теряются
- Проверь что localStorage включен
- Очисти кэш браузера
- Попробуй другой браузер

### Telegram бот не работает
- Проверь что бот создан в @BotFather
- Проверь что URL правильный
- Попробуй /start команду
