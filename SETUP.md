🚀 СПОСОБ 1: GITHUB PAGES (РЕКОМЕНДУЕТСЯ - БЕСПЛАТНО)

   ШАГ 1: Создать GitHub репозиторий
   ────────────────────────────────
   
   1. Перейди на github.com
   2. Создай новый репозиторий с именем: hook-butcher
   3. Скопируй SSH ссылку
   
   ШАГ 2: Загрузить файлы
   ────────────────────────
   
   git clone git@github.com:ТВОЙ_НИКИ/hook-butcher.git
   cd hook-butcher
   
   # Скопируй файлы dist/ сюда
   cp -r dist/* ./
   
   git add .
   git commit -m "Deploy Hook Butcher v2.0"
   git push origin main
   
   ШАГ 3: Включить GitHub Pages
   ────────────────────────
   
   1. Перейди в Settings репозитория
   2. Pages → Source: main branch
   3. Сохрани
   
   ГОТОВО! Твоя игра доступна на:
   https://ТВ НИКИ.github.io/hook-butcher/

🔧 СПОСОБ 3: СОБСТВЕННЫЙ СЕРВЕР

   Если используешь nginx:
   
   /etc/nginx/sites-available/hook-butcher:
   ───────────────────────────────────────
   
   server {
       listen 80;
       server_name hook-butcher.com;
       root /var/www/hook-butcher/dist;
       
       location / {
           try_files $uri $uri/ =404;
       }
   }
   
   Скопируй файлы dist/ в /var/www/hook-butcher/dist/
   
   sudo systemctl restart nginx
   
   Готово! Доступно на: https://hook-butcher.com

📱 ИСПОЛЬЗОВАНИЕ TELEGRAM MINI APP (НЕСКОЛЬКО СПОСОБОВ)

   СПОСОБ 1: Через кнопку в боте
   ──────────────────────────────
   
   В @BotFather для твоего бота:
   1. Edit → Menu Button
   2. Выбери "Web App"
   3. Введи URL игры
   
   Теперь пользователи видят кнопку "Играть" в чате

   СПОСОБ 2: Через команду
   ──────────────────────
   
   Отправь боту команду:
   /setcommands
   
   game - Запустить игру
   
   Команда скопирует этот JavaScript в бота:
   
   // Когда пользователь пишет /game:
   bot.onMessage('/game', (msg) => {
       bot.sendWebApp({
           url: 'https://ТВ_URL/hook-butcher/'
       });
   });

   СПОСОБ 3: Через inline кнопку
   ────────────────────────────
   
   Бот может отправлять:
   
   Привет! 🎮
   [Играть в Крюк Мясника]
   (кнопка открывает минипроект)
