СПОСОБ A: БЕЗ ПРОГРАММИРОВАНИЯ (Самый простой - 5 минут)

   1. Открой @BotFather в Telegram
   
   2. Отправь команду: /newbot
      • Имя бота: Hook Butcher 🔪
      • Username: hook_butcher_bot (или уникальное)
      • Получишь TOKEN
   
   3. Отправь команду: /setmenubutton
      • Выбери только что созданного бота
      • Выбери Web App
      • Введи URL: https://ТВ_URL.github.io/hook-butcher/
      • Текст кнопки: 🔪 Играть
   
   4. Откончательно готово!
      • Пользователи видят кнопку "Играть" в чате с ботом
      • При клике открывается твоя игра

СПОСОБ B: С ПРОСТЫМ BACKEND (Node.js)

   Требуемо: Node.js 16+

   ШАГ 1: Создай файл bot.js

   const TelegramBot = require('node-telegram-bot-api');
   const TOKEN = 'ТВОЙ_TOKEN_ОТ_BOTFATHER';
   const bot = new TelegramBot(TOKEN, {polling: true});

   bot.on('message', (msg) => {
       const chatId = msg.chat.id;
       
       if (msg.text === '/start') {
           bot.sendMessage(chatId, 'Добро пожаловать в Крюк Мясника! 🔪', {
               reply_markup: {
                   inline_keyboard: [[
                       {
                           text: '🔪 Играть',
                           web_app: {url: 'https://ТВ_URL/'}
                       }
                   ]]
               }
           });
       }
   });

   console.log('Bot running...');

   ШАГ 2: Установить зависимости
   
   npm install node-telegram-bot-api

   ШАГ 3: Запустить бота
   
   node bot.js

FAQ ДЛЯ TELEGRAM

   Q: Бот требует выделенный сервер?
   A: Нет для Web App. Если делаешь backend - нужен сервер или serverless

   Q: Сколько стоит?
   A: Бот бесплатный. Хостинг Telegram Mini App - бесплатный

   Q: Могу добавлять звуки/видео?
   A: Да! Всё работает как в обычном браузере

   Q: Как получить Telegram Stars для платежей?
   A: Подай заявку в @BotFather, потом /setinvoice
